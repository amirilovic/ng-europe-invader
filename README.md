ng-europe-invaders
==================

ngEurope 2014 Hackathon Game

[![ngEurope Invader Preview](http://img.youtube.com/vi/rphMHL05K5w/0.jpg)](http://www.youtube.com/watch?v=rphMHL05K5w)


Setup
```
$ npm run setup
```

Start

```
$ npm start
```
